# 安鼠之亂

> 「鼠患正在失控。市民、毛孩、貓頭鷹和老鷹的命運，全在您的槌子（與毒餌）之上。」

一款用 Phaser 4 + Rsbuild + TypeScript 寫的諷刺類網頁打地鼠小品。
玩家扮演一位虛構的城市首長，在 5 大鼠跡熱區奮力出槌、努力保住政治生涯。

⚠️ **本遊戲純屬虛構，若有雷同純屬巧合。** 任何角色、地名、毒餌公斤數、市政公告、政論台詞⋯⋯都是隨手寫的，與任何現實中的人事時地物無關。

---

## 玩法簡介

- 老鼠 🐭 從洞口冒出來，請敲下去
- 貓 🐱、狗 🐶、貓頭鷹 🦉、黑鳶 🦅 也會冒出來，**請不要敲下去**（會扣分、會被市民罵）
- 集滿老鼠藥 💊 之後，按 **空白鍵**（手機是 **三指觸控**）可以無差別清場
  - 一次清光全螢幕、清得很爽
  - 連帶把場上的毛孩跟天敵也一起送走
  - 身為一個好市長，是不是不要用比較好呢？
- 場上同時老鼠太多 → **漢他病毒爆發** → Game Over，市民不開心、議員打電話來

五大熱區由輕到重：**萬華 → 大同 → 中正 → 中山 → 大安**。最後一關是大安魔王關，請繫好安全帶。
